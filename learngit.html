Git的安装和使用

一、安装，自报家门：

安装后Git后，打开Git-Bash，输入

注意 git config 命令的 --global 参数，用了这个参数，表示你这台机器上所有的 Git 仓库都会使用这个配置，当然也可以对某个仓库指定不同的用户名和 Email 地址。

二、创建版本库：


三、提交文件：

git add readme.txt ---->将readme.txt文件添加到暂存区
git commit -m "描述" ------>提交

四、时光穿梭机：

1、查看仓库状态：git status

表示修改了内容但是还没提交。
2、查看修改了哪些内容(未提交的前提下)： git diff
3、查看修改的历史记录： git log


4、返回上一个版本： git reset --hard HEAD^（一个 ^ 代表一个版本，也可 HEAD~100）
5、查看内容：cat readme.txt
6、回到未来的某个版本： git reset --hard ID
7、记录每一次命令： git reflog,可以查看每一次的ID
8、 撤销修改(丢掉工作区的修改)：git checkout --readme.txt

9、把暂存区的修改撤销掉：git reset HEAD readme.txt
10、删除文件：git rm readme.txt ,再提交

六、从远程库克隆到本地 ：

git clone git@github.com:WenZhongWei/gitskills.git

七、分支：

1、创建分支：git branch dev
2、切换到某个分支：git checkout dev
3、以上可以合并为：git checkout -b dev
4、查看现有的分支：git branch

5、合并指定分支到当前分支：git merge dev
6、删除分支：git branch -d dev
7、查看分支的合并情况：git log --graph --pretty=oneline --abbrev-commit
8、合并时禁用Fast forward：git merge --no-ff -m"merge with no-ff" dev
